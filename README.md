## wsui

#### 主要功能
###### 图片查看，支持图片放大、缩小、翻转等功能

#### 使用方法
###### 安装
> npm i wsui

###### 初始化引入
```
import Wsui  from 'wsui'
Vue.use(Wsui)
```

###### 页面中使用
```
<WsuiIcon type="home" :round=true />
<WsuiInput :require=true />
```
或
```
<wsui-input :require=true />
<wsui-button>确认</wsui-button>
```
